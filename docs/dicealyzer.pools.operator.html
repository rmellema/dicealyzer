dicealyzer.pools.operator module¶

The module for dice pools consisting of other different dice pools. It also installs operators on AbstractDicePool to make it easier to combine dice pools in Python.

class dicealyzer.pools.operator.BinaryOperatorPool(operator, operator_str, *operands)[source]¶

Bases: dicealyzer.pools.base.AbstractDicePool

A class to represent multiple dice pools that are combined by an operator. This operator is assumed to work with the python function reduce.
