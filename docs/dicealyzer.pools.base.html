dicealyzer.pools.base module¶

The base module for all dice pools.

class dicealyzer.pools.base.AbstractDicePool[source]¶

Bases: abc.ABC

The abstract base class for all dice pools. Defines several convinience methods based on the abstract methods presented.

abstract roll()[source]¶

Roll the dice in this pool, and return the result. Also returns the values of all rolled dice as a list.

pretty_string()[source]¶

Print the dice pool in a pretty manner.

abstract property values¶

All the possible values that this dice pool can produce, as a set. Ascending order is preferred, but not required.

abstract probability(value)[source]¶

Returns the probability of a certain value being rolled.

Alias for probability

prob_less_equal(value)[source]¶

Give the probability that this dice pool rolls equal to or lower than value.
