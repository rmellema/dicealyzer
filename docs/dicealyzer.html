Module contents¶

This module provides ways to define and combine pools of dice so they can be analyzed. It consists of a submodule for creating and analyzing dice pools and a module for reading in dice pools from strings.

dicealyzer.compare(*dpools)[source]¶

Compare a group of pools on the values they can produce and their probabilities. Returns a dictionary from numbers to a list of numbers. The keys are the values that the pools together can produce, and the values are the probabilities that the pools will produce those values, in the same order as the input.
