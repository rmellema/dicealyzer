===========
diceroller
==========
This project is a toy project to play around with the `Lark parser`, since I
felt like it. It can read in a command in `standard dice notation`, including
modifiers and multiple dice seperated by `+` and give back the roll.

.. _Lark parser: https://github.com/lark-parser/lark
.. _standard dice notation: https://en.wikipedia.org/wiki/Dice_notation
